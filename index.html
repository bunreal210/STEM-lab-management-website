Cài Đặt Telegram Bot

📖 Hướng dẫn lấy Bot Token & Chat ID:

  1. Mở Telegram → tìm @BotFather → gõ /newbot → đặt tên → lấy Bot Token
  2. Nhắn tin /start cho bot vừa tạo
  3. Truy cập: api.telegram.org/bot{TOKEN}/getUpdates để lấy Chat ID (trường id trong from)
